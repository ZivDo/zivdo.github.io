movies={1:"Django", 2:"Basterds", 3:"Pulp", 4:"Avatar", 5:"Titanic"}
这货定义的方式与list定义不同之处在于：
它不用[]，而是用{}。对于里面每一个元素来说，还是以逗号分开。
不过你可能感觉到奇怪的是1:"Django"这个是什么意思
很简单，冒号前面的1表示索引即关键字，冒号后面的"Django"表示索引到的值。
也就是说 我调用movies[1]所得到的值是 Django
那么 movies[2]肯定是 Basterds了
这样你可能就回有疑问，那根list有什么区别呢？
区别在于 我可以把 冒号前面的数字 换成 字符串，这样 索引的话，就要比以前的好记多了，不用记下标了。

关键字: break, continue, pass

break 和 continue一般会用在循环语句中，
break:如果在程序中遇到break，是指退出该循环
continue:如果在程序中遇到continue，是指退出本次循环，继续进行下一次循环

pass:pass语句是指不做任何事情

详细文档请见:http://docs.python.org/3/tutorial/controlflow.html#break-and-continue-statements-and-else-clauses-on-loops

例子:
break:
for i in range(10):
	if i == 8:
		break
	else:
		print( i )

输出结果
0 1 2 3 4 5 6 7

continue:
for i in range(10):
	if i == 8:
		continue
	else:
		print( i )

输出结果
0 1 2 3 4 5 6 7 9

主要使用在 当条件不满足或者满足的时候，不做任何事情，又由于if:语句里面不能为空语句，所以用pass来代替

i = 11
if i <= 10:
	pass
else:
	print( i )

No.2 

冒泡算法
numbers = [ "8", "6", "3", "9", "5", "2", "4", "1", "7" ]

冒泡排序算法的思路如下：
1.比较相邻的元素。如果第一个比第二个大，就交换他们两个。
2.对每一对相邻元素作同样的工作，从开始第一对到结尾的最后一对。在这一点，最后的元素应该会是最大的数。
3.针对所有的元素重复以上的步骤，除了最后一个。
4.持续每次对越来越少的元素重复上面的步骤，直到没有任何一对数字需要比较。

最后输出结果应该是:[ "1","2","3","4","5","6","7","8","9" ]

Tips: 推荐使用while进行循环，并且要使用数组下标
